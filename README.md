# RL_Ensemble
RL model to predict actions for stock trading


Instructions:
1. “brew install python@3.12”

     Run the following command “python3.12 —version” to see the version installed through    home-brew.


2.  “Cd” to your working directory and clone the repository
          “git clone  https://github.com/abinayg/RL_Ensemble.git “

3. You should see a folder “RL_Ensemble” in your current working directory


4. Setup the python virtual environment
         “python3.12 -m venv <env_name>”

5. Activate the virtual environment
         “ source <path_to_env>/bin/activate “

6. Install the required modules from the requirements.txt
         “pip install -r ~/RL_Ensemble/requirements.txt”

7. Open Visual Code Studio and select this virtual environment kernel.

8. Run the “final_script.ipynb” and if everything is followed correctly, you should be able to see no issues in running the notebook.
